# 🌿 Vedic Human – AI Powered Yoga Trainer
In House Project

## 📌 Overview
**Vedic Human** is an AI-based web application designed to help users practice yoga correctly with real-time feedback. It combines traditional Vedic wellness practices with modern technologies like Artificial Intelligence and Computer Vision.

The system analyzes user posture using AI pose detection and provides accuracy scores, correction suggestions, and performance tracking.

---

## 🎯 Objectives
- Promote healthy lifestyle through yoga  
- Provide real-time posture correction using AI  
- Make yoga training accessible and affordable  
- Track user performance and improvement over time  

---

## 🚀 Features
- 🔐 User Login & Authentication  
- 🧘 Yoga Session with Webcam  
- 🤖 AI Pose Detection (MoveNet / MediaPipe)  
- 📊 Real-time Feedback & Accuracy Score  
- 📈 Progress Tracking & Session History  
- 💻 Simple and User-Friendly Interface

## 🔄 Working

1. User logs into the system  
2. Starts a yoga session  
3. Webcam captures pose data  
4. AI model analyzes posture  
5. System generates feedback & accuracy  
6. Data is stored for progress tracking
